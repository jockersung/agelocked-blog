## 前言

我們常常看到或聽到「本產品 XXXXX 經 SGS 檢驗合格，100 % 純天然，讓您吃得安心！用的放心！」這樣就能代表販售商品的「品質」嗎？

當然，有報告總比沒報告好，至少廠商願意花錢去證明安全是一件好事，但是這樣就要證明塗抹在臉上甚至吃進去的東西安全還是遠遠的不足。

為什麼這樣說：

**因為即使 SGS 檢驗的樣本符合檢驗出來的結果，但是廠商生產的產品不見得與樣本相同。**

**而且也有可能出現廠商沒有提到的項目，而是與他宣稱的效果相反甚至有害的。**

另外一個原因是產品的安全性，不單單只有檢查的那些項目，而是從頭到尾完整的「**生產製程** 」。

如果全部都通過的話，才能說這個產品的品質安全合格。

## **什麼是 SGS**

這要從SGS 歷史說起。

SGS 是**瑞士通用公證行** （ Société Générale de Surveillance S.A.，簡稱 SGS S.A.）的一般簡稱，是一家成立於西元 1878 年，總部設在瑞士日內瓦，是**專門**替國際商品** 提供測試**（Testing）、**檢驗** （Inspection）及**認證** （Certification）服務的跨國集團。

目前是世界上**最大** 在國際貿易中有影響的民間獨立檢驗機構，一年的商品檢驗業務量占世界貿易總量的 5 %。

公司可進行各種物理、化學和冶金分析，包括進行破壞性和非破壞性試驗，向委託人提供一套完整的數量和品質檢驗以及有關的技術服務；

如提供裝運前的檢驗服務；

提供各種與國際貿易有關的諸如商品技術、運輸、倉儲等方面的服務；

監督跟購銷、貿易、原材料、工業設備、消費品遷移有關聯的全部或任何一部分的商業貿易暨操作過程的服務。

## **還有其他家檢驗公司嗎？**

世界上有很多像這樣的**第三方檢驗** （Third Party Inspection，TPI）公司，各有其善長處。

在台灣有設立分公司的有法商—法立德（Bureau Veritas，BV）公證有限公司、英商—全國公證（Intertek）檢驗股份有限公司、德商—德國萊茵（TÜV Rheinland ）技術監護顧問股份有限公司、美商—通用檢驗科技股份有限公司（GSL Lab），而台灣本土則有台美檢驗公司（Super Lab）、汎美檢驗（ACCURO LAB）等等，為品質把關。

## **SGS 有哪些項目** ？

  * **認證** （Accreditation）

指主管機關對某實驗室或驗證機構給予正式認可，證明其有能力執行某特定工作。

例如：SGS的食品服務部是經過衛生福利部TFDA認證。

❗️注意：SGS 並不是主管機關，所以當消費者看到商品標示「本產品通過SGS認證」，其實是非正確用語！  

  * **驗證** （Certification）

由中立之第三者確認某一項產品、過程或服務，能否符合規定要求達到一定標準。

例如：SGS 的驗證暨企業優化事業群就是一個驗證機構，稽核員到欲取得 ISO 9001 的公司去稽核，便是執行一項驗證的工作；若是這些公司符合相關的要求，通過SGS稽核員的審核，便能取得 ISO 9001 的驗證證書。

因此通過稽核審查的品牌店家或企業廠商可以告知消費者「本公司通過SGS ISO 9001驗證」。

此外，SGS 也提供 CE、環保標章、SGS Q Mark …等產品驗證服務，但「產品驗證」與「只送樣品到實驗室測試」相比要複雜嚴謹許多。

❗️注意：產品驗證除了測試樣品外，還要進行文件審查或現場稽核以確保其品質水準，因此若店家僅有送樣品進行測試，是不可以對消費者宣稱「此產品通過 SGS 驗證」!  

  * **測試** （Testing）

指在實驗室依照標準程序進行樣品的分析。

例如：廠商將產品的樣品送至 SGS 測試，SGS 會根據相關的測試規範（常常是國際標準或是 CNS 標準）進行測試，並簽發正式的測試報告（Test Report）告知委託者測試結果。

❗️注意：由於測試報告僅就委託者送來的樣品負責，若SGS未參與抽取樣品，該測試報告就無法代表整批貨品的品質。

若僅送樣品到實驗室測試卻宣稱產品經過驗證，可能會讓消費者誤解其所有產品批號都有一定的品質，不可不慎。

  * **檢驗** （Inspection）

通常指在工廠或倉庫，依照買方要求進行商品的目視檢查與專業判斷是否可以出貨，但因受限於設備與時間，大多無法進行精密的儀器分析及判定。

## **小測驗檢視差異**

( ❌ ) 本產品通過SGS認證！安全無毒！(若是產品驗證應說明獲得哪一種標章)  
( ⭕️ ) 本公司○○產品送至SGS測試，報告編號為xxxxxxxx。  
( ⭕️ ) 工廠系統ISO 22000經SGS驗證通過。  
( ⭕️ ) 本公司食品/ 原物料定期委託 SGS食品服務部測試。  
( ❌ ) 本公司某產品經過SGS檢驗合格，品質保證。 (若僅送某項樣品測試，SGS無法保證該產品之所有品質項目)

## SGS 檢驗報告如何看

不是只有提供報告就好，看懂報告才是真的了解。基本上大略分為這四項：

  * **誰驗的**

(1) 執行檢測的公司名稱：SGS 或是其他檢驗公司

(2) a.號碼：獨一無二，相當於報告的身分證號碼（不同檢驗公司的編碼方式不同）

b.日期：報告日期與現在時間點愈接近，參考性愈高。

**提供的報告日期若已超過該批次產品的有效期限，測試結果就沒有參考價值**

c.頁碼：透過頁數可以判斷這是不是一份完整報告，例如一項產品有好幾種檢測，完整報告共有三頁，有些廠商可能只挑合格的部分提供，或者張冠李戴，拿別人的檢驗報告掐頭去尾假裝自己的提供給消費者。

而且報告最後一頁呈現被測試的樣品照片也是幫助您確認測試結果是指那個產品。

  
(3) a.送樣廠商：申請檢測的廠商名稱、廠商地址/聯絡人等。

b.產品資訊：測試報告一定會附上產品名稱、樣品包裝、樣品狀態、收樣日期與測試日期，而產品批號、產品型號、製造日期及有效日期的資訊有最佳。

(4) 有廠商戳章：如 SGS 會提供廠商一份正本、兩份副本及電子檔案，都有 SGS 原戳章，才不會被竄改。

SGS 報告01

  * **驗甚麼項目**

(5) 測試需求：送樣廠商所指定檢測方式，如 ASTM F963 為美國標準組織「ASTM國際標準組織」發佈的玩具標準  
(6) 測試項目：此次檢測的主要項目，廠商會根據產品的潛在風險做不同測項之把關，如：肉品主要風險測項為動物用藥；即食食品為衛生指標菌或病原菌；茶葉或農作物為農藥殘留等。

  * **用甚麼檢驗方法**

(7) 測試方法：列出檢測方法來源（如 ASTM F963）及檢測儀器名稱（如感應耦合電漿原子發射光譜儀 ICP-AES）。

測試方法通常以該類產品的主管機關公告方法或國家標準 CNS 方法為主，其次是國際公告方法，最後才是參考文獻。

  * **檢驗結果**

(8) 測試結果：  
a. MDL(方法偵測極限值)：測試儀器能檢測的最低門檻(如磅秤最低只能量到 0.1 公克，小於 0.1 公克就量不出來)  
b. 結果：實際檢測出來的數值；如 MDL 為 2 mg/kg，小於 2 就會顯示為”n.d.”未檢出。  
C. 限值：檢測的法規或檢測標準的最高容許限值。

SGS 報告02

**看備註就知道，這個檢驗報告只對廠商提供的「樣品」做測試，且針對廠商「有付錢的那幾項內容」檢測，沒付錢的就不在範圍內。**

**SGS 也不會保證樣品和業者正在販售的商品是不是一樣或是有沒有違法問題。**
